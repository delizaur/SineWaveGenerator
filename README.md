# Sine Wave Generator
A design for a sine wave generator using an AT89 microcontroller that integrates a DDS (Direct Digital Sythesis) chip, USB, LCD display and matrix keyboard. Made for the Microcontroller Systems (EMISY) course at the Warsaw University of Technology (Politechnika Warszawska).

### Parts
- [AT89C5131A-S3SUM Microcontroller](https://www.digikey.com/en/products/detail/microchip-technology/AT89C5131A-S3SUM/1008478)
- HY1602E LCD Display
- AD9837 Direct digital synthesis chip
- Membrane matrix keypad
